read_cr_rate()¶

Method for reading in the incident cosmic ray rate.

This method will generate a pandas.DataFrame containing the incident cosmic ray rate for each observations, as well as all the associated metadata generated by utils.metadata.GenerateMetadata.

For metadatum containing an array of values (i.e. latitude, longitude , and altitude) the pandas.DataFrame will have two columns. One column will represent the value at the start of the observation, while the other will represent the value at the end of the observation (e.g. altitude_start and altitude_end).

Finally, the pandas.DataFrame generated is indexed using a pandas.DatetimeIndex to facilitate time-series anaylses.

read_cr_stat(fill_value=- 999, units=None, min_exptime=200)¶

Read in all the data for the specified statistic

This method should only be used to read in the following statistics:

  • Electron deposition [electrons]

  • Size [pixels]

  • Size [sigmas]

  • Shape [dimensionless]

To read in the incident cosmic ray rate, see read_cr_rate()

Parameters

  • fill_value (int) – If there are any NaNs, replace them with this value. The default is -999

  • units ({'pixels', 'sigmas'}) – Specifies the units for the sizes statistics.

Returns

Populate the given statistics corresponding attribute with a masked dask.array

Return type

None
