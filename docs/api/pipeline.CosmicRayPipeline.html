run_labeling_all(chunk_num)¶

Run the labeling analysis and compute the statistics

Run the labeling process to extract data for every CR in each image and save the results.

Parameters

Returns

  • runtime (float) – Time required to process in minutes

  • results (tuple) – Results from analyzing all files in flist.
