CosmicRayLabel¶

class label.labeler.CosmicRayLabel(fname, gain_keyword=None)¶

Bases: label.labeler.Label

Class for generating the cosmic ray label

Parameters

  • fname (str) – Name of FITS file

  • gain_keyword (str) – Keyword to use for extracting gain conversion. Currently only the average gain across all readout amplifiers is used for the conversion of DN to ELECTRONS. Additionally, the conversion is only applied if the BUNIT keyword is DN or COUNTS.

run_ccd_label(deblend=False, use_dq=True, extnums=[1, 2], threshold_l=None, threshold_u=None, plot=False)¶

Run labeling algorithm on CCD data

This will populate the following class attributes:

  • dq

  • integration_time

  • label

  • sci

Parameters

  • deblend (bool) – If True, deblend overlapping cosmic rays

  • extnums (list) – List of the extension numbers. This should always be a list, even if it just contains one element

  • use_dq (bool) – If True, generate the label using the DQ information

  • threshold_l (int) – Objects found that affect fewer pixels than this limit are removed

  • threshold_u (int) – Objects found affect more pixels than this limit are removed
