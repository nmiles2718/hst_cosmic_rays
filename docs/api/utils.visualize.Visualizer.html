  • i (int) – Integer used to determine the color of the points used in the scatter plot

    Type of smoothing to apply to the cosmic ray rate dataset.

    String alias for the time period representing the size of the moving window. Some common ones are listed below:

    • ’15D’, 15 day window

    • ’M’, one month window

  • min_periods (int) – Minimum number of datapoints that must be in a given window
